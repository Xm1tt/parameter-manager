Parameter Manager — Privacy Policy

Version 1.0 · Effective date: 2026-06-17 · Publisher: xm1tt (Gustavo Schmitt)
Contact: parameter.manager.support@gmail.com

Summary

Parameter Manager is a desktop add-in for Autodesk Inventor. It processes your Inventor documents and configuration files locally on your computer. We do not collect, transmit, sell, or share personal data. We do not use analytics, advertising, cloud services, or tracking technologies in this application. The application can modify your Inventor documents when you choose to apply changes; please read the Data modification and Disclaimer sections below.

Scope

This Privacy Policy applies to:

  • Parameter Manager software distributed via the Autodesk App Store and other channels authorized by the publisher
  • The publisher website page that hosts this policy
  • Support communications you initiate by email

This policy does not govern Autodesk Inventor, the Autodesk App Store, or third-party websites you open from links inside Parameter Manager.

Information we do not collect

Parameter Manager v1.0 does not:

  • Require user accounts or registration
  • Collect names, email addresses, or other contact information through the app
  • Collect usage analytics or telemetry
  • Send data to publisher servers or cloud backends
  • Use cookies, web beacons, or similar tracking in the application
  • Access the Internet except when you choose to open a link (for example, Privacy Policy or support email)

Information processed locally

The application processes the following only on your device:

DataPurposeStorage
Inventor parameters, assemblies, and components Display, edit, stage, and apply changes you request Inventor documents (via Autodesk Inventor APIs)
Preset CSV files and JSON metadata Preset library, diff, and staging Folders you configure in Settings
Configuration packages and recommendation rules Library browse, preview, and authoring Folders you configure in Settings
Application settings (paths, theme, filters, column layout) Restore your preferences Local JSON/settings files on your computer
Temporary diagnostic logs (optional) Troubleshooting local issues %TEMP%\ParameterManagerUI.log, %TEMP%\ParameterManagerAddIn.log

You choose library folder paths in Settings. Those folders may contain project-related data you place there; the publisher does not access them remotely.

About the diagnostic logs: the optional log files may contain technical details such as file paths, document and parameter names, add-in actions, and operation results, recorded to help diagnose issues. These logs are stored only on your computer in the %TEMP% folder and are never transmitted automatically. You can delete them at any time. If you choose to send logs to support, review them first and remove anything confidential.

Data modification and your responsibility

Parameter Manager is a productivity tool that can modify your Inventor documents when you choose to apply changes (for example, editing parameter values, expressions, comments, names, or applying suppress/unsuppress operations). Changes are staged first and are only written to the model when you confirm an Apply action.

  • Save and back up your work before applying changes. Although Apply uses a single Inventor undo transaction, you remain responsible for maintaining backups of your files.
  • Review the results after each Apply, especially on large or interdependent parameter sets.
  • You are responsible for the configuration files, presets, and packages you create, import, or share.

Children

Parameter Manager is not directed to individuals under 18 years of age. We do not knowingly collect personal information from anyone.

Third parties

Autodesk Inventor
Parameter Manager runs inside or alongside Inventor and uses Inventor APIs to read and modify documents you open. Inventor’s handling of data is governed by Autodesk’s Privacy Statement.

Autodesk App Store
If you download Parameter Manager from the Autodesk App Store, Autodesk may process information related to the download and your Autodesk account under Autodesk’s policies. That processing is separate from Parameter Manager’s local operation.

Microsoft Windows
The application uses standard Windows APIs (file system, UI, optional shell folder picker). No publisher cloud service is involved.

Data retention and deletion

  • Inventor document changes follow Inventor’s save/undo behavior.
  • Library files remain in folders you configured until you delete or move them.
  • Temporary log files in %TEMP% can be deleted by you at any time.
  • Uninstalling Parameter Manager does not automatically delete your library folders or Inventor documents.

Security

Parameter Manager is designed for local use without a publisher backend. You are responsible for securing your computer, Inventor files, and library folders. Keep Inventor and Windows updated.

If you contact support, do not send confidential business data unless necessary to reproduce an issue.
