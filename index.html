Parameter Manager for Autodesk Inventor

Edit, organize, reuse, and apply parameter changes across assemblies and components — with presets, configuration packages, and a safe stage-then-apply workflow.

Version 1.0 · Inventor 2026–2027 (64-bit) · Publisher: xm1tt
All processing runs locally on your computer.

What it does

  • Browse, filter, and edit parameters in a high-performance grid
  • Stage changes, review pending edits, then apply in a single Inventor undo transaction
  • Manage preset libraries with diff preview before staging
  • Author and preview configuration packages and recommendation rules
  • Work alongside Inventor — the main window does not block the CAD interface

Documentation

Usage Instructions Installation, ribbon access, grid workflow, presets, packages, and support. Privacy Policy Local-only processing. No accounts, analytics, or cloud data collection.
