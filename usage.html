Parameter Manager — Usage Instructions

Version 1.0 · Autodesk Inventor 2026–2027 (64-bit) · Publisher: xm1tt
Support: parameter.manager.support@gmail.com

Overview

Parameter Manager is an Inventor add-in that helps you browse, filter, edit, stage, and apply parameter changes across assemblies and components. It also provides a configuration library for presets, configuration packages, and recommendation rules. All processing runs locally on your computer; the add-in does not send your model data to publisher servers.

Requirements

  • Autodesk Inventor 2026 or 2027 (64-bit)
  • Windows 10 or 11 (64-bit)
  • An assembly or part document open for parameter editing

Installation

  1. Download Parameter Manager from the Autodesk App Store.
  2. Install the application bundle (administrator privileges may be required).
  3. Restart Inventor if it was open during installation.
  4. Open Tools → Add-Ins and confirm Parameter Manager is loaded and not blocked.

No manual file copying or registry editing is required when installing from the App Store.

Opening Parameter Manager

  1. Open an Inventor assembly (recommended) or part.
  2. Go to the Assemble tab.
  3. In the Parameter Manager panel, click Parameter Manager.

The main window opens as a companion to Inventor. Click the ribbon button again to focus an already open window. While the ribbon tooltip is visible, press F1 to open this guide (or the bundled Quick Start PDF).

Main grid workflow

Browse and filter

  1. Use the search box and filter checkboxes (Modified, Equations, Key, Renamed, Consumed, type filters).
  2. Switch view mode: Flat, Group by Parameter, or Group by Component.
  3. Click column headers to sort. Expand or collapse groups as needed.

Stage edits (nothing is written yet)

  1. Edit values, expressions, or comments directly in the grid.
  2. To rename parameters: edit the Name column for User and Reference parameters. Model parameters that were previously renamed in Inventor can also be renamed.
  3. Staged rows show pending status badges. Changes are not written to Inventor until you apply.
  4. When you stage a change, the row is typically auto-selected for apply.

Apply to Inventor

  1. Select the rows you want to commit (only selected rows with pending changes are applied).
  2. Click Apply on the bottom bar.
  3. Inventor updates the model in a single undo transaction.
  4. Review the Apply summary dialog for successes and any failures.
  5. Use Inventor Undo (Ctrl+Z) to revert if needed.

Click the ? button on the apply bar for a short in-app workflow summary.

Presets

Presets are reusable sets of parameter values stored as CSV files in your library folder.

  1. Click Configuration… on the main toolbar to open the Configuration Browser.
  2. Open Preset Library.
  3. Browse presets, review the diff, and optionally add presets to the queue.
  4. Click Stage to grid to load preset values into the main grid.
  5. Return to the main window, review staged rows, and click Apply.

Configure the library and presets folders in Settings before first use.

Configuration packages

Packages are ordered configuration files that can include parameter changes and structural operations.

  1. In the Configuration Browser, select a package and click Review package.
  2. In Configuration Package Preview, review Overview, Parameters, Components, and Recommendations.
  3. For parameters: click Stage parameters to grid, then Apply on the main window.
  4. For structural changes (suppress/unsuppress): click Apply suppress… in the Preview window.

Note: Parameter apply and structural apply use separate actions in v1.0.

To create a new package, use New package… in the Configuration Browser.

Settings

Click Settings on the main toolbar to configure:

  • Configuration library root folder
  • Presets folder
  • Default filters, view mode, and visible columns
  • Theme and window opacity
  • Whether to keep the window attached to Inventor

Point library folders to a writable location before using presets or packages. Open Privacy Policy from Settings for data handling information.

Uninstall

  1. Remove Parameter Manager via the Autodesk App Store / ApplicationPlugins uninstall flow, or delete the bundle folder from %ProgramData%\Autodesk\ApplicationPlugins\.
  2. Restart Inventor.

Library files in folders you configured in Settings are not deleted automatically.
